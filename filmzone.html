Filmzone

Filmzone is a Movie Database Website built with React, using themoviedb.org API. Designed in Adobe XD.

  • Live Site
  • Github

Tools Used

  • React
  • SCSS
  • JSX
  • Rest API
  • Adobe XD

The Requirements

This requirements of this project was to create a movie database website using React, while pulling movie data using the themoviedb.org Rest API to display on this site. This website should be mobile responsive and well styled. This project was to be completed in a group of 3.

The Challenge

Importing the react-multi-carousel into the project was fairly straightforward, but it took a little more dissecting to understand how to target specific areas of the carousel to style it according to our design was slightly more difficult, especially without breaking the responsiveness of the carousel.

The Outcome

This project was a one of the larger sites we created and I realized how working in React is so much easier to organize and work with smaller components, especially when working with a team. This project also taught us how to efficiently use Git and Github for version control. React is incredibly robust, and I plan to create more sites using React (and pairing it with headless WordPress) for future sites.
