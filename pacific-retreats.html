Pacific Retreats

This is a cabin rental site created with WordPress and WooCommerce. Low-Fidelity and High-Fidelity prototypes were created in Figma.

  • Live Site
  • Github

Tools Used

  • WordPress
  • WooCommerce
  • Rest API
  • Figma

The Requirements

This project required our group of 4 to build an e-commerce site for a small tourism vacation rental business. The customers should be able to book their accommodation and pay online. Design is an important in terms of aesthetic of the location and should target the proper target demographic. The site should use appropriate plugins and Rest APIs to emulate a real site we would make for a client.

The Challenge

Setting up the Maps JavaScript API key was a little more confusing than expected, as there were many different APIs, setting options, and restrictions available, but I was able to obtain the appropriate API key and use it with a third-party WordPress plugin to output the Google Map on our site.

The Outcome

We started the project by researching and deciding what type of content we want to include in our site, designs and inspiration that we wanted, and then creating a site map and development plan for Pacific Retreats. We created a low-fidelity and high-fidelity prototype of the site in Figma, and developed the site in WordPress. Google Map Rest API was used to output the locations for our cabins, and Instagram API used to output our Instagram posts on the home page.
